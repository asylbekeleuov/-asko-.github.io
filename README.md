## 📌 Жоба туралы

**АСЖ AI** — қазақстандық мұғалімдерге арналған жасанды интеллект көмекшісі. Пәннің ерекшелігін, оқу бағдарламасын және сыныптың деңгейін ескере отырып, **5 минутта** толыққанды сабақ жоспарын, дидактикалық материалдар мен презентация құрылымын жасайды.

## 🚀 Мүмкіндіктер

| Мүмкіндік | Сипаттама |
|-----------|-----------|
| 📋 Сабақ жоспары | 45 минуттық толыққанды жоспар (SMART мақсаттармен) |
| 📚 Дидактикалық материал | А/Б/В деңгейлі тапсырмалар, флэш-карточкалар |
| 🖥️ Презентация | 12-15 слайд құрылымы + дизайн ұсыныстары |
| 🌈 Инклюзивтілік | Ерекше білім беру қажеттіліктері бар оқушыларды қолдау |
| ⬇️ Жүктеу | TXT форматында барлық материалды жүктеу |

## 📋 Кіріс өрістері

- **Пән** — Математика, Биология, Тарих және т.б.
- **Сынып** — 1-11 сынып
- **Сынып бағыты** — Жаратылыстану, Гуманитарлық, Жалпы білім және т.б.
- **Тақырып** — Нақты сабақ тақырыбы
- **Кітап баспасы** — Мектеп, Атамұра, Алматыкітап және т.б.
- **Инклюзивті оқушы** — Бар/жоқ + ерекшелік түрі

## 🛠️ Технологиялар

- **Frontend:** HTML5, CSS3 (Custom Properties, Flexbox, Grid), Vanilla JS
- **AI:** Google Gemini 1.5 Flash API
- **Fonts:** Unbounded (Google Fonts), Noto Sans
- **Deploy:** GitHub Pages compatible (статикалық файлдар)

## ⚡ Орнату және іске қосу

### 1. Репозиторийді клондаңыз

```bash
git clone https://github.com/yourusername/asj-ai.git
cd asj-ai
```

### 2. Файлдарды ашыңыз

```bash
# Тікелей браузерде ашу
open index.html

# Немесе Live Server (VS Code) арқылы
```

### 3. Gemini API кілтін алыңыз

1. [Google AI Studio](https://aistudio.google.com/app/apikey) сайтына кіріңіз
2. "Create API Key" батырмасын басыңыз
3. Сайттың API кілт өрісіне енгізіп, "Сақтау" батырмасын басыңыз

## 🌐 GitHub Pages-те жариялау
